OGC Web Testing Facility for NSF CINERGI

This site helps validate metadata instances, as part of the NSF project Community Inventory of EarthCube Resources for Geosciences Interoperability (CINERGI). It is based on the OGC Web Testing Facility.

The code used in this project is hosted at GitHub:

  • TEAM Engine - Java Testing Engine.
  • ETS 19139 Test - Code in TestNG used to execute the test.

If you need more information please contact the Director of the Compliance Program, Luis Bermudez.
